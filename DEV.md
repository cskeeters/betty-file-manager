# Update Dependencies

```sh
go get -u
go mod tidy
```

# Create Release

1. Create a tag in the git repo and push to github.

    ```sh
    git tag v0.4.0
    git push origin v0.4.0
    ```

2. Click on the tag button on the github page.

3. Click "..." next to the uploaded tag.

4. Select *Create Release*
